### Запуск проекта для Windows

1-Запустить проект в IDE, например, в WebStorm

2 - Установить все зависимости и плагины
```
npm install
```
3 - Создать в проекте .env
```
copy example.env .env
```
4 - Вставить токен от бота в .env

5 - Запустить бота
```
npm start
```
